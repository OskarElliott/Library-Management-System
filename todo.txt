Use of Generative AI

During development of this project, I used ChatGPT to help generate an initial version of a synthetic data seeding script for testing purposes. I provided the database schema, project requirements, and data generation specifications, and Claude produced a draft implementation.

The generated code was reviewed, tested, and modified by me to ensure that it matched the final database schema and project requirements. All debugging, validation, integration with the project, and final design decisions were completed by me.

Prompt used (summarised):
"Generate a Python script that populates my SQLite library database with realistic synthetic data according to the provided schema and statistical requirements."

The generated script served as a starting point and was subsequently adapted for use in the final system.
